1
Xcode 설치 마무리
App Store 다운로드 완료 후 진행

Xcode를 한 번 실행해서 추가 구성요소(Command Line Tools 등)를 자동 설치하게 한 뒤, 터미널에서 아래 두 명령어를 순서대로 실행합니다.

① 개발자 도구 경로 설정
sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
② 라이선스 동의
sudo xcodebuild -license accept
🔑
두 명령어 모두 Mac 로그인 비밀번호를 물어봅니다. 입력 시 화면에 아무것도 표시되지 않는 게 정상이에요 (보안 처리).
2
CocoaPods 설치
iOS 의존성 관리자 — 현재 설치되어 있지 않음

Homebrew를 통해 CocoaPods를 설치합니다. 터미널에서 실행하세요.

brew install cocoapods
📌 Homebrew가 없다면 먼저 brew 설치가 필요합니다. 오류가 나면 알려주세요.
3
아이폰 준비
USB 연결 및 개발자 모드 활성화
  • USB 케이블로 아이폰을 Mac에 연결합니다.
  • 아이폰 잠금을 해제하면 "이 컴퓨터를 신뢰하시겠습니까?" 팝업이 뜹니다 → 신뢰를 탭합니다.
  • (아이폰) 설정 → 개인정보 보호 및 보안 → 개발자 모드 → 켬 → 재부팅합니다.
4
서명(Signing) 설정 — 무료 Apple ID 등록
처음 한 번만 진행하면 됩니다

Xcode GUI에서 서명을 설정하는 것이 가장 확실합니다. 아래 순서대로 진행하세요.

  • 먼저 pod 설치를 위해 터미널에서 실행합니다 → ios/app.xcworkspace 파일이 생성됩니다.
cd ios && pod install && cd ..
  • 생성된 워크스페이스를 Xcode로 엽니다.
open ios/app.xcworkspace
  • Xcode 상단 메뉴 Xcode → Settings → Accounts → + 버튼 → Apple ID 로그인
    무료 계정도 OK — 로그인하면 "Personal Team"이 자동 생성됩니다
  • Xcode 왼쪽 파일 트리에서 app 타겟을 클릭 → Signing & Capabilities 탭으로 이동합니다.
  • "Automatically manage signing" 체크박스를 활성화합니다.
  • Team 드롭다운에서 본인의 Personal Team을 선택합니다.
5
빌드 & 아이폰에 설치
첫 빌드는 5~15분 소요됩니다

아래 명령어를 실행하면 연결된 기기 목록이 표시됩니다. 본인의 아이폰을 선택하세요.

npx expo run:ios --device
⏱
첫 빌드는 5~15분이 걸립니다. 이후 빌드는 훨씬 빠릅니다.
빌드 중 오류가 발생하면 오류 메시지를 그대로 공유해 주시면 같이 디버깅하겠습니다.
6
아이폰에서 개발자 신뢰
첫 설치 후 한 번만 — "신뢰되지 않은 개발자" 오류 시

앱 아이콘은 설치됐는데 실행 시 "신뢰되지 않은 개발자" 메시지가 뜨면 아래를 따라하세요.

  • (아이폰) 설정 → 일반 → VPN 및 기기 관리 진입
  • 본인 Apple ID를 탭합니다.
  • 신뢰를 탭합니다.
✅
설정 후 앱 아이콘을 탭하면 정상 실행됩니다.
⚠️
미리 알아두어야 할 2가지
🗓
무료 계정은 7일마다 앱 서명이 만료됩니다.
7일 후 앱이 열리지 않으면 npx expo run:ios --device만 다시 실행하면 됩니다. 코드나 설정을 다시 건드릴 필요 없어요.
🔐
로그인(카카오/구글)이 안 될 경우 Supabase 설정을 확인하세요.
Supabase 대시보드 → Authentication → URL Configuration → Redirect URLs에
duckjillog://** 가 등록되어 있어야 로그인 콜백이 정상 작동합니다. 없으면 추가해 주세요.
▶
두 번째부터 — JS만 새로고침할 때

빌드(네이티브 코드)를 건드리지 않고 JavaScript만 수정한 경우, 빌드 없이 빠르게 확인할 수 있습니다.

npx expo start --dev-client

아이폰에서 앱을 열면 자동으로 연결됩니다.
