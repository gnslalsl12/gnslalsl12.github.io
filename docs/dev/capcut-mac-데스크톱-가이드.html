Pricing

요금제 4단계로 개편

무료 / Standard($9.99) / Pro($19.99·₩19,800) / Team($24.99)으로 재편. Mac에서 4K 브이로그를 다룬다면 사실상 무료 또는 Pro 둘 중 하나입니다.

Features

무료로도 충분히 강력

Shortcuts

Mac은 Cmd 중심

Windows의 Ctrl 자리에 Cmd. 핵심 4개는 Space · ⌘B · Delete · ⌘Z. 여기에 J/K/L 셔틀만 더하면 됩니다.

무료 내보내기 제한 메모. Pro 자산(프리미엄 템플릿·AI 효과)을 쓴 무료 사용자는 주당 export가 제한(보고에 따라 3~6회)되고 초과 시 워터마크가 붙습니다. 다만 이는 공식 단일 정책이 아니며, 순수 무료 자산만 쓴 편집은 횟수 제한 없이 워터마크 없이 내보내진다는 보고도 있습니다. 웹 버전은 제한이 없다는 보고도 있어 진화 중인 정책으로 보는 게 맞습니다.

03

Mac 데스크톱 기본 기능

카테고리별로 정리한 핵심 편집 도구.

Timeline

타임라인 편집

분할(Split), 트림, 좌/우 삭제(Q/W), 속도 조절·속도 곡선, 멀티트랙, 컴파운드 클립·그룹화.

Text

텍스트·자막

수동 텍스트, 자동 자막(AI 음성 인식 후 동기화), 스타일 자막, SRT 내보내기, 다국어 번역.

Transitions

전환·필터·효과

Smooth·Glitch·Zoom·Wipe 전환, 필름·드리미·글리치 효과, 프리셋 필터.

Audio

오디오 편집

무료 음악·사운드 이펙트, 노이즈 제거, 보컬 분리, 오디오 추출, 페이드·볼륨 키프레임, 비트 동기화.

AI Tools

AI 기능

오토컷, 배경 제거(Auto Cutout), 크로마키, 모션 트래킹, 오토 리프레임, 손떨림 보정, TTS, AI 아바타.

Color

색보정

컬러휠, 커브, HSL, LUT 임포트, AI 컬러 보정, 컬러 매치, 화이트밸런스.

Keyframe

키프레임 애니메이션

위치·스케일·회전·불투명도·색상 애니메이션. Alt+K로 패널 표시, 속도 곡선과 결합.

Format

고해상도 푸티지

4K·GoPro·드론 푸티지 지원. 멀티트랙으로 B-roll·내레이션·음악을 레이어링.

04

여행 브이로그에 특히 유용한 기능

현장 촬영본을 살리는 도구 모음.

01

손떨림 보정 (Stabilizer)

핸드헬드·걷기 영상용. Recommended / Minimum cut / Most stable 모드 제공.

02

오토 리프레임

16:9 → 9:16 등 플랫폼별 자동 재구성. 안정화·카메라 이동 속도 조절 포함.

03

비트 동기화

음악 비트에 컷을 맞춰 리드미컬한 몽타주를 빠르게 완성.

04

색보정·LUT

시네마틱 룩, 여러 클립 컬러 매치로 영상 전체의 톤을 일관되게 유지.

05

자동 자막 + TTS

현장 음성이 없는 장면에 내레이션과 자막을 자동으로 입힘.

06

멀티트랙 레이어링

B-roll·내레이션·텍스트·음악을 별도 트랙으로 분리해 구조적으로 편집.

06

정책·약관 (2026 확인 결과)

소유권과 데이터 관련 배경 정리.

  • CapCut은 ByteDance 소유. 2025년 1월 미국에서 TikTok과 함께 일시 삭제됐다가 행정명령으로 복귀했습니다.
  • 2026년 1월 22일 TikTok USDS 합작법인 거래가 종결되며 CapCut도 동일한 세이프가드 하에 들어가 법적 압력이 완화됐습니다. 2026년 중반 현재 미국에서 사용 가능합니다.
  • 인도에서는 2020년 6월부터 영구 금지 상태가 유지됩니다.
!

약관 논쟁. 2025년 6월 발효된 약관을 두고 ByteDance에 광범위한 콘텐츠 사용 권리를 부여한다는 분석이 제기됐습니다.

이에 대해 CapCut은 사용자 생성 콘텐츠 조항이나 콘텐츠 사용 방식에는 수년간 변화가 없었다고 공식 반박했습니다. 양측 입장이 상충하므로, 민감한 클라이언트 작업 시에는 이 논쟁을 인지하고 완성본을 별도 보관하는 편이 안전합니다.
