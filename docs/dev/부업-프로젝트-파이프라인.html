꼭 기억할 것

  1. 견본은 복사해서 쓴다. design/도, 문서도, 에이전트도 전부 복사본 안에. 원본엔 넣지 않는다.
  2. progress.md는 프로젝트 루트에. 안 그러면 슬래시 커맨드가 헛돈다.
  3. 에이전트 세팅 후 세션 재시작. 재시작 전엔 새 에이전트가 안 보인다.
